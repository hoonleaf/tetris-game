FastAPI + SQLite 백엔드를 구축해 회원/점수 데이터를 관리한다.

이메일+비밀번호 회원가입/로그인을 구현해 로그인한 사용자만 점수를 기록하도록 한다.



게임이 끝날 때마다 사용자의 게임 점수를 저장하고, 전체 사용자 중 최고 점수를 프론트에서 표시한다.

전체 구조



프론트엔드: 이미 작성된 테트리스 UI/로직을 사용하는 index.html / game.js에 로그인/회원가입 UI 및 API 연동 코드를 추가.



백엔드: backend/main.py (예시) 에 FastAPI 앱, SQLite 연동, 인증/점수 API 라우트 구현.



DB: SQLite 파일 (예: tetris.db) 에 사용자/점수 테이블 정의.
